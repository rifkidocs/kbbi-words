# 📖 KBBI Word Search

Aplikasi pencarian kosa kata KBBI (Kamus Besar Bahasa Indonesia) yang cepat, cerdas, dan modern. Dirancang untuk membantu pengguna menemukan kata-kata Bahasa Indonesia berdasarkan awalan tertentu dengan performa yang dioptimalkan.

🌐 **Live Demo:** [https://kbbi-words.vercel.app/](https://kbbi-words.vercel.app/)

## ✨ Fitur Utama

- **Pencarian Instan:** Hasil muncul secara real-time saat Anda mengetik atau menekan Enter.
- **Dataset Luas:** Menggunakan database KBBI V dengan lebih dari 112.000+ kata unik.
- **Optimasi Prioritas:** Algoritma pencarian yang memprioritaskan kata-kata dengan akhiran unik (x, y, z, q, f, v, w) untuk kebutuhan khusus (seperti permainan kata).
- **Desain Modern:** Antarmuka berbasis Bento Grid yang bersih, responsif, dan mendukung Mode Gelap (Dark Mode).
- **SEO Optimized:** Konfigurasi metadata lengkap untuk visibilitas mesin pencari dan media sosial yang lebih baik.

## 🚀 Teknologi yang Digunakan

- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router)
- **Library UI:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Bahasa:** [TypeScript](https://www.typescriptlang.org/)
- **Dataset:** Dataset kosa kata dalam format JSON yang efisien.

## 🛠️ Instalasi Lokal

1. Clone repositori ini:
   ```bash
   git clone https://github.com/username/kbbi-word-search.git
   ```
2. Masuk ke direktori proyek:
   ```bash
   cd kbbi-word-search
   ```
3. Instal dependensi:
   ```bash
   npm install
   ```
4. Jalankan server pengembangan:
   ```bash
   npm run dev
   ```
5. Buka [http://localhost:3000](http://localhost:3000) di browser Anda.

## ⚖️ Hak Cipta dan Kepemilikan Data

Dataset dalam proyek ini diekstraksi dari referensi aplikasi mobile KBBI. Seluruh data dimiliki sepenuhnya oleh:

> **Badan Pengembangan dan Pembinaan Bahasa**  
> **Kementerian Pendidikan, Kebudayaan, Riset, dan Teknologi**
